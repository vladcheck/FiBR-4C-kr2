# E-commerce API

## Запуск

В данный момент поддерживается только ручной запуск сервера и клиента через терминал.

### Бекенд

```bash
cd /src/server # из корня проекта
npm install
npm run start
```

После этого в консоли будут указаны URL, по которым можно обратиться к API, а также посмотреть Swagger (`/api-docs`). Сервер будет доступен по порту `3000`.

### Фронтенд

```bash
cd /src/frontend # из корня проекта
npm install
npm run dev
```

Клиент будет доступен по порту `3001`.

## Маршруты

### Клиент

| Маршрут                                | Пояснение                                 |
| :------------------------------------- | ----------------------------------------- |
| `/`                                    | Главная страница                          |
| `/shop`                                | Каталог товаров                           |
| `/products/:id`                        | Страница товара                           |
| `/register`                            | Регистрация                               |
| `/login`                               | Логин                                     |
| `/profile`                             | Информация о профиле данного пользователя |
| `*` (Любые другие маршруты ведут сюда) | Страница 404                              |

### Сервер

| Маршрут              | Методы   | Нужна авторизация? | Разрешенные роли |
| :------------------- | -------- | ------------------ | ---------------- |
| `/`                  | `GET`    | Нет                | Все              |
| `/api-docs`          | `GET`    | Нет                | Все              |
| `/api/products`      | `GET`    | Нет                | Все              |
| `/api/products/:id`  | `GET`    | Да                 | Все              |
| `/api/products/:id`  | `PUT`    | Да                 | seller           |
| `/api/products/:id`  | `DELETE` | Да                 | seller           |
| `/api/users`         | `GET`    | Нет                | Все              |
| `/api/users/:id`     | `GET`    | Нет                | Все              |
| `/api/users/:id`     | `DELETE` | Да                 | Все              |
| `/api/auth/register` | `POST`   | Нет                | Все              |
| `/api/auth/login`    | `POST`   | Да                 | Все              |
| `/api/auth/refresh`  | `POST`   | Да                 | Все              |
| `/api/auth/me`       | `GET`    | Да                 | Все              |
